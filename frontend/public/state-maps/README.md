# State District Map SVGs

Add district map SVG files here for interactive state maps.

**Naming:** `{state-slug}.svg` (e.g. `uttar-pradesh.svg`)

**Source:** Download from [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:SVG_maps_of_Uttar_Pradesh) (public domain).
- Uttar Pradesh: [Districts_of_Uttar_Pradesh.svg](https://commons.wikimedia.org/wiki/File:Districts_of_Uttar_Pradesh.svg)

Each SVG should have `<path>` elements with `id` or `aria-label` matching district names (e.g. `Pratapgarh`, `Agra`).
